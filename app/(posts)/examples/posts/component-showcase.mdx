# Preview Component

The Preview component allows you to showcase your components with a live preview.

# Usage

```tsx
<Preview>
  <YourComponent />
</Preview>
```

# Features

- Live preview of components
- Syntax highlighting
- Copy code button
- Responsive design

# Customization

You can customize the preview by adding props to the Preview component:

```tsx
<Preview codeblock={true}>
  <YourComponent />
</Preview>
```

# Best Practices

1. Keep components focused and single-purpose
2. Use TypeScript for better type safety
3. Add proper documentation
4. Include examples in your documentation
